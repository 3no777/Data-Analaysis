✈️ Flight Delays Analysis
Ky projekt analizon vonesat e fluturimeve duke përdorur Python, Pandas dhe Matplotlib.
Qëllimi i projektit është të studiohen shkaqet kryesore të vonesave dhe të paraqiten rezultatet me grafikë.

📌 Çfarë përfshin projekti


Leximi i dataset-it të fluturimeve


Pastrimi i të dhënave


Analiza statistikore


Krijimi i grafikëve



🛠 Teknologjitë e përdorura


Python


Pandas


Matplotlib



📂 Dataset
Dataset-i i përdorur:
flight_delays_dataset.csv
Dataset-i përmban të dhëna rreth:vonesave të fluturimeve
motit
problemeve teknike
fluturimeve të anuluara
trafikut ajror

📊 Analizat e realizuara
1. Bar Chart
Paraqet vonesën mesatare sipas shkakut:


Carrier Delay


Weather Delay


NAS Delay


Security Delay


Late Aircraft Delay


2. Histogram
Shfaq shpërndarjen e vonesave (ArrDelay).
3. Scatter Plot
Analizon lidhjen midis numrit të fluturimeve dhe vonesës totale.
4. Line Plot
Paraqet vonesën mesatare sipas muajve.

📈 Rezultatet kryesore


Carrier Delay dhe Late Aircraft Delay janë shkaqet kryesore të vonesave.


Security Delay ka ndikimin më të vogël.


Fluturimet më të ngarkuara kanë më shumë vonesa.


Disa muaj kanë nivel më të lartë vonesash se të tjerët.
